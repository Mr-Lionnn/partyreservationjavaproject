# partyreservationjavaproject

Membre du groupe: 


Projet: 

L'objectif de ce projet est de réaliser une plateforme web conviviale de réservation en ligne de salles de fête en utilisant Java avec Spring Boot, tout en explorant les principaux concepts comme les controllers web, la persistance des données avec JPA/Hibernate, et une interface utilisateur simple avec les JSP, l’amélioration des IHM avec le framework Boostrap. Une base de données relationnelle (Postgresql, Mysql…) sera utilisée pour la partie persistance.
